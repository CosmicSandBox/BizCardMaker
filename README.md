# 🦁 내 생애 첫 명함!

## 🎈 About Project

1. 기획 배경

- 멋쟁이사자처럼 11기 모집을 홍보할 목적으로 만들게 됐습니다.

2. 핵심 기능

- 이름, 학과 등 정보를 입력하고 디자인을 선택하면 명함이 만들어집니다.
- 3D로 확인할 수 있고 이미지로 다운로드 받을 수 있습니다.
- 카카오톡, 페이스북 등 다양한 SNS로 공유할 수 있습니다.

3. 기대 효과

- 예비 멋쟁이사자처럼 11기분들에게 들어오면 어떠한 활동을 할 수 있는지 대략적으로 보여줄 수 있습니다.
